BiruniMap — The Global Map of the Iranian Diaspora

  • Role

    Founder & CEO

  • Since

    2024 – present

  • Live platform

    birunimap.com

A full-stack web platform connecting Iranians worldwide through an interactive map of businesses, cultural events, and community knowledge — built entirely from scratch, solo, in my spare time.

The Problem

There are millions of Iranians living abroad — in Europe, North America, Australia, and beyond. When they move to a new city, they face the same invisible challenge: finding their community.

Where is the Iranian restaurant that serves real ghormeh sabzi? Which doctor speaks Farsi? Is there a Nowruz celebration nearby? The answers exist — buried in WhatsApp groups, scattered across Facebook posts, passed by word of mouth — but there is no single place to find them.

For newcomers especially, this gap is not just inconvenient. It is isolating.

My Solution

I built BiruniMap — a bilingual (English and Persian), interactive world map that brings the Iranian diaspora together in one place. The platform currently lists over 3,000 verified Iranian-owned businesses across multiple continents, alongside a live events calendar and a community blog.

The name is a tribute to Abu Rayhan al-Biruni, the 10th-century Persian polymath who travelled the world and spent his life mapping human civilisation. The spirit of his work — curiosity, respect for every culture, and the belief that knowledge belongs to everyone — is exactly what BiruniMap is built on.

I designed and developed the entire platform myself: from database architecture and backend APIs to the frontend interface and the map experience. The tech stack includes Next.js and TypeScript on the frontend, PHP 8 on the backend, MySQL as the database, Leaflet.js for interactive mapping, and Tailwind CSS for the UI. The platform is deployed on a cPanel server with GitHub Actions handling continuous delivery via FTP. Authentication is implemented with JWT, and the system supports role-based access with admin approval workflows for both businesses and events.

Beyond the technical build, I manage the community side of the platform — coordinating volunteers who help verify listings, curating events, and growing the user base organically. BiruniMap is free to use, community-supported, and built with the long-term goal of being the go-to resource for every Iranian living abroad, anywhere in the world.
